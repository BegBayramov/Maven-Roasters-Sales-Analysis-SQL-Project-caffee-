# Maven-Roasters-Sales-Analysis-SQL-Project-caffee-
This project analyzes transactional sales data from Maven Roasters, a fictional coffee shop with multiple locations. 
The goal of the analysis is to explore sales performance, customer behavior, and product insights using SQL.
